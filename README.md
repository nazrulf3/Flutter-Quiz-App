## Getting Started

### Prerequisites

- Flutter SDK (3.0 or higher)
- Dart SDK
- Android Studio / VS Code
- iOS Simulator / Android Emulator

### Installation

1. Clone the repository

```bash
git clone https://github.com/nazrulf3/quiz_app.git
```

2. Navigate to project directory

```bash
cd quiz_app
```

3. Get dependencies

```bash
flutter pub get
```

4. Run the app

```bash
flutter run
```
